Top 15 IntelliJ IDEA shortcuts

1. Alt + Enter
 : Alt + Enter 키보드 단축키를 사용하면 무엇이든 수정할 수 있다.

2. F2
 : 제안을 탐색할 때 사용할 수 있다. F2를 활용해 다음 오류, 경고, 제안으로 이동해보자.
F2를 통해 수정이 필요한 곳으로 이동하고, Alt + Enter를 통해 수정할 수 있다.

3. Alt + 1(Windows)
 : 프로젝트 창 열기

4. Esc
 : 프로젝트 창을 연 후, 에디터로 초점을 옮기고 싶으면 Esc를 누르면 된다.
변경 사항을 적용하지 않고 모든 팝업을 닫는 데 유용하다.

5. Ctrl + E
 : Ctrl + E를 사용해 가장 최근 열었던 파일을 확인해보자.

6. Ctrl + B(Windows)
 : 코드 안에서 다른 코드로 이동하고 싶을 때 Ctrl + B를 사용해 심볼의 선언으로 이동해보자.
Ctrl + Alt + B(Windows)를 사용하면 구현을 탐색하는 것도 가능하다.

7. Alt + F7
 : 종종 선언을 검색하는 대신 사용 위치를 검색하고 싶은 경우, Alt + F7를 누르면
모든 사용 위치가 표시된다.

8. Ctrl + Ctrl
 : 어디서든 Ctrl을 두 번 누르면 무엇이든 실행 가능하다. Run Anything 창을 열어준다.

9. Ctrl + W, Ctrl + Shift + W(Windows)
 : 커서 주변의 코드 섹션을 확장하거나 축소할 수 있다. 

10. Ctrl + /
 : 주석을 추가할 수 있다. Shift + Ctrl + / 를 눌러 블록 주석을 추가할 수 있다.

11. Shift + Ctrl + Enter(Windows)
 : 현재 구문 완성 기능은 코딩 할 때에 가장 유용한 단축키에 해당합니다.

12. Ctrl + Alt + L(Windows)
 : 간편하게 현재 파일의 서식을 프로젝트 표준에 맞춰보자. 
다른 범위를 선택하여 서식을 지정하려면,  Shift + Ctrl + Alt + L(Windows) 을 누르고 
원하는 옵션을 선택하자. 예를 들어, 전체 파일의 서식을 다시 지정할 수도 있다. 

13. Shift + Ctrl + Alt + T(Windows)
 : InterlliJ IDEA 리팩토링의 대부분은 개별 단축키로 지정되어 있지만, 
Shift + Ctrl + Alt + T(Windows)를 사용하면 모든 리팩토링을 표시하는 것도 가능하다. 

14. Shift + Ctrl + A(Windows)
 : 해당 단축키를 누르면 IntelliJ IDEA의 모든 액션을 검색할 수 있다. 
드롭다운 메뉴에는 액션 뿐 아니라 단축키도 함께 표시되므로 단축키를 익히고 
연습할수도 있다. 

15. Shift + Shift
 : 전체 검색 기능이다. 검색 창에는 기본적으로 최근 파일이 표시되므로 
Ctrl + E 대신 이 창을 활용할 수 있다. 검색어를 입력하면 클래스, 파일, 심볼 및 
액션의 검색 결과가 모두 표시된다.
